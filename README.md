# Python-Mini-Project

This repository contains two beginner-friendly Python projects created for learning and practice purposes.

##  Projects Included

### 1️⃣ Password Generator
A simple Python program that generates a random password based on user-defined length.

#### Features
- User enters password length
- Generates random password
- Uses letters and numbers
- Easy to understand


### 2️⃣ ATM Simulation
A console-based ATM simulation program using Python.

#### Features
- Enter initial balance
- Check balance
- Deposit money
- Withdraw money
- Menu-driven program


## Technologies Used
- Python
